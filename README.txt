Inside this project i will try to show you possibility of using JTabbedPane component.

This component helps to represent data from various number of panels without changing sizes of them or of our JFrame or JPanel.

Also this compoment could be used as a unusual way of representing information like bookmarks from the book , in this way we try to show panels as a pages from the book.

Graphical components have been created using WindowBuilder as a instulled component for Eclipse Mars.

Technologies :
Java
WindowBuilder(component for Eclipse)
